#Plataforma e-Commerce - Implementación de Clases Básicas (PHP + AJAX)

##Descripción del Proyecto

Este proyecto corresponde a la Asignación No. 2 – Implementación de Clases Básicas para una Plataforma e-Commerce.  

Se desarrolló una aplicación web utilizando Programación Orientada a Objetos (POO) en PHP, implementando las clases fundamentales de un sistema e-Commerce: Producto, Usuario y Carrito de Compras.  

La aplicación integra comunicación asíncrona mediante AJAX (Fetch API en JavaScript) para consumir una API desarrollada en PHP.

---

##Tecnologías Utilizadas

- PHP 8 (Programación Orientada a Objetos)
- JavaScript (ES6)
- AJAX (Fetch API)
- HTML5
- XAMPP (Servidor Apache local)
- Git & GitHub
